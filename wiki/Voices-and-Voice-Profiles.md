You can group multiple voice profiles under a single **Speaker**.

- **Assign to Speaker**: Link a profile to a speaker identity.
- **Variant Name**: Give the profile a variant (e.g., "Whisper", "Angry").
- **Default Profile**: Set which profile under a speaker should be used by default in the Performance tab.
